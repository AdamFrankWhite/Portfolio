The Wonder of JQuery

This is an exciting post - to know enough about JavaScript to be able to appreciate the utility of JQuery.

JQuery is one of the most popular JavaScript libraries. It serves two main purposes: 1) It provides extensive cross-browswer support. 2) It simplifies so much JavaScript code with shorthand.

Take, for example, an AJAX request. In vanilla (plain) JavaScript, it would take this many lines of code to make a request:

var request = new XMLHttpRequest(); request.open('GET', 'someaddress.com/file.json'); request.onload = function () { var results = JSON.parse(request.responseText) console.log(results) } request.send();
